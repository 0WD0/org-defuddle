* Framework Performance Benchmarks
:PROPERTIES:
:AUTHOR: Jordan Lee
:SITE: Jordan Lee
:WORD_COUNT: 117
:END:

We measured the response latency of several popular frameworks under realistic workloads. Here are the results:

| Framework | Latency (ms) | Year | Runtime | Notes |
| [[https://example.com/alpha][Alpha]] | 12 | 2019 | Native | *Fastest overall* |
| [[https://example.com/bravo][Bravo]] | 25 | 2020 | JIT | Uses *aggressive caching* |
| [[https://example.com/charlie][Charlie 2.0]] | 40 | 2018 | Interpreted | Legacy support only |
| Delta *optimized* | 55 | 2021 | Native | Requires [[https://example.com/config][custom config]] |
| [[https://example.com/echo][Echo]] | 90 | 2017 | VM | *Most popular* in surveys |
| Foxtrot | 150 | 2022 | Interpreted | Still in *beta* |

The fastest frameworks tend to use native compilation, while interpreted runtimes show higher latency across the board.
